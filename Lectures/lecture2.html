JavaScript Basics

Script Tag

The script tag is used to include JavaScript code in an HTML document. It can be placed in the head or body section of the document. The code can be inline or linked to an external file using the src attribute.
Variables
var x = 5; We dont want to use this anymore, it is outdated. let y = 10; This is the new way to declare variables. const z = 15; This is used for constants that should not change.
Inline Script
<script> console.log("Hello, World!"); </script>
External Script
<script src="script.js"></script>

JavaScript Control Structure

Iterative Statements

For Loop
The for loop is used to execute a block of code a specific number of times. It consists of three parts: initialization, condition, and increment/decrement. for (let i = 0; i < 5; i++) { console.log(i); }
While Loop
while (condition) { // code to be executed }
Do-While Loop
The do-while loop is similar to the while loop, but it guarantees that the code block will be executed at least once before checking the condition. do { // code to be executed } while (condition);

Functions

Arguments
Access argument list with arguments keyword. function myFunction() { console.log(arguments[0]); // Access the first argument }
function adder(...values) means that the values parameter is always a non empty array.
Closure
A function within a function

Events

Key Event Properties:
  1. target: the element that triggered the event
  2. cancelable: tells you if an event can be canceled or not. Read-only
  3. type: tells you the name of an event, like 'click', 'submit', 'mouseover'

Custom Events

Creating Custom Events
You can create custom events using the CustomEvent constructor. const myEvent = new CustomEvent('myEvent', { detail: { key: 'value' } });
Dispatching Custom Events
You can dispatch custom events using the dispatchEvent method. document.dispatchEvent(myEvent);
