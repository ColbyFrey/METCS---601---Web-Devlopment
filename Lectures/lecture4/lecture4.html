Flexbox

You must use a parent element for your items inside the flex container

Flex Direction

flex-direction, controls if you want a row or a column

  • column vs column-reverse: reverse your column elements

Flex Wrap

The default is to not wrap any items

flex-wrap, controls if you want your items to wrap or not

  • nowrap: items will not wrap
  • wrap: items will wrap to the next line
  • wrap-reverse: items will wrap to the next line in reverse order

flex-flow>

Shorthand for flex-direction and flex-wrap

Flex shorthand

Shorthand property to set individual

  • flex-grow: how much the item will grow relative to the other items
  • flex-shrink: how much the item will shrink relative to the other items
  • flex-basis: the initial size of the item before it grows or shrinks

CSS Grids

Flexbox is pretty one-dimensional items, good for ui-components

CSS Grids are a powerful layout system that allows you to create complex layouts with ease.

By default grids are a single column. Each child will get its own row!

If the parents have a fixed height the children will be equally sized

Grid Container

  • grid-template-columns: defines the columns of the grid
  • grid-template-rows: defines the rows of the grid
  • grid-gap: defines the gap between the grid items
  • grid-area: defines the area of the grid item

Media Queries

Media queries are used to apply styles based on the screen size or device type.

They are rules for specific screen sizes, when the rule inside the @media() tag hits it will run

  • @media (max-width: 600px) { ... } - applies styles when the screen width is less than 600px
  • @media (min-width: 600px) { ... } - applies styles when the screen width is greater than or equal to 600px
  • @media (orientation: portrait) { ... } - applies styles when the device is in portrait mode

Grid-gap (Old) vs gap (new)

grid-gap is the old way of defining the gap between grid items, gap is the new way.

gap is a shorthand property that sets both row-gap and column-gap at the same time.

grid-auto-flow

grid-auto-flow is a property that controls how the grid items are placed in the grid.

  • row: items are placed in rows
  • column: items are placed in columns
  • dense: items are placed in the grid as soon as there is space available

Pseudo Classes and Elements

Pseudo classes are used to style elements based on their state or position in the document.

  • :not()
  • ::first-letter || ::first-line - Selects the first letter or line in the text
  • ::before || ::after - Used to insert content before or after an element
    • p::before { content: "Hello "; }
  • ::selection - allows you to style selected text on a page
